# Hesabdari System

## Overview

The Hesabdari System is an accounting management solution designed for mobile and computer repair shops. It aims to simplify the financial processes involved in managing repair orders, customer data, and overall business performance. This documentation provides an in-depth look at the features, functionalities, and usage of the system.

## Features

- **Customer Management**: Easily store and manage customer information, including contact details and repair history.
- **Repair Order Tracking**: Keep track of all repair orders, including status updates and completion dates.
- **Invoicing**: Generate and send invoices to customers based on services rendered.
- **Inventory Management**: Manage parts and products needed for repairs, including stock levels and supplier information.
- **Financial Reporting**: Access comprehensive reports on income, expenses, and profit margins to aid in financial decision-making.

## Installation Instructions

1. **Clone the Repository**: Use the following command to clone the repository to your local machine:
   ```bash
   git clone https://github.com/saeedrezaie98-ctrl/hesabdari-system.git
   ```

2. **Install Dependencies**: Navigate to the project directory and install any required dependencies using:
   ```bash
   npm install
   ```

3. **Start the Application**: Launch the application with:
   ```bash
   npm start
   ```

## Usage

- Navigate to the application in your web browser at `http://localhost:3000`.
- Create a new customer by filling out the customer form.
- Add a new repair order by selecting the customer and entering the details of the repair.
- Generate invoices through the invoice section.

## Technologies Used

- **Frontend**: React.js
- **Backend**: Node.js and Express.js
- **Database**: MongoDB
